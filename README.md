Designed for remote collection of data for academic research on Human-Robot Interactions (HRI).
Contains a description of the study, its purpose, sample inclusion/exclusion criteria.
Pre-task collects basic demographic details & responses on a standardized scale known as the Interpersonal Reactivity Index (IRI).
Main task is a simple point-and-click task that poses the participant with hypothetical moral dilemmas and asks them to choose between one of 4 anthropomorphic robots.
